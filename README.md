# OOP-SHAPES
# machineTools-OOP
The OOP-shapes exercise demonstrates the features of object oriented programming in a real life scenerio of shapes and how their areas are calculated.
This is an OOP challenge from the Andela bootcamp to test skills on the full meaning and properties of OOP.
These properties include:
* Inheritance
* Encapsulation
* Polymorphism
* Abstraction
